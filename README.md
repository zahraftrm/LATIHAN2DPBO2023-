## ALUR

Membuat 3 atribut multi-level inheritance. Ketiga atribut tersebut berisi:
- Human (atribut nenek)
- Sivitas Akademik (atribut anak)
- Mahasiswa (atribut cucu)

Kemudian terdapat satu file main dan semua data di input secara hardcode.
